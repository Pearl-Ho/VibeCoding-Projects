# 贪吃蛇 - Greedy Snake Game

用 Python 和 Pygame 做的贪吃蛇小游戏，这是我的第一个 Python 项目。

## 功能
- 蛇身渐变绿色，吃了食物会有变长效果
- 顶部实时显示得分、长度
- 每吃5个食物速度+1，越玩越快
- 开局3秒倒计时，吃到食物有屏幕闪光反馈

## 运行方法
pip install pygame
python 贪吃蛇.py

## 开发工具
- Python 3.11
- Pygame 2.6
- TraeCN 和 Claude 协作完成

# Greedy Snake Game

Snake game made with Python and Pygame. This is my first Python project.

## Function
The snake's body gradually turns green, and eating food will cause it to grow longer.
Real-time display of score and length at the top.
Speed +1 for every 5 pieces of food eaten, the more you play, the faster you get
3-second countdown at the start, and screen flash feedback when eating food.

## Run method
pip install pygame python greedy snake.py
